# [圆圈中最后剩下的数字](https://leetcode-cn.com/problems/yuan-quan-zhong-zui-hou-sheng-xia-de-shu-zi-lcof/)

约瑟夫环

```java
class Solution {
    public int lastRemaining(int n, int m) {
        if (n == 0) return -1;
        if (n == 1) return 0;
        return (lastRemaining(n - 1, m) +  m) % n;
    }
}
```
