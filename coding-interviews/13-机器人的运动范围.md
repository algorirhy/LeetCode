# [机器人的运动范围](https://leetcode-cn.com/problems/ji-qi-ren-de-yun-dong-fan-wei-lcof)

### 方法一

DFS

```java
class Solution {
    int m, n, k;
    boolean[][] visited;

    public int movingCount(int m, int n, int k) {
        this.m = m;
        this.n = n;
        this.k = k;
        this.visited = new boolean[m][n];
        return dfs(0, 0, 0 ,0);
    }

    private int dfs(int x, int y, int sumX, int sumY) {
        if (x >= m || y >= n || k < sumX + sumY || visited[x][y]) {
            return 0;
        }
        visited[x][y] = true;
        int newSumX = (x + 1) % 10 == 0 ? sumX - 8 : sumX + 1;
        int newSumY = (y + 1) % 10 == 0 ? sumY - 8 : sumY + 1;
        return 1 + dfs(x + 1, y, newSumX, sumY) + dfs(x, y + 1, sumX, newSumY);
    }
}
```

### 方法二

BFS

```java
class Solution {
    public int movingCount(int m, int n, int k) {
        int res = 0;
        boolean[][] visited = new boolean[m][n];
        Deque<int[]> queue = new ArrayDeque();
        queue.add(new int[]{0, 0, 0, 0});
        while(!queue.isEmpty()) {
            int[] tmp = queue.remove();
            int x = tmp[0], y = tmp[1], sumX = tmp[2], sumY = tmp[3];
            if (x >= m || y >= n || k < sumX + sumY || visited[x][y]) continue;
            visited[x][y] = true;
            res++;
            int newSumX = (x + 1) % 10 == 0 ? sumX - 8 : sumX + 1;
            int newSumY = (y + 1) % 10 == 0 ? sumY - 8 : sumY + 1;
            queue.add(new int[]{x + 1, y, newSumX, sumY});
            queue.add(new int[]{x, y + 1, sumX, newSumY});
        }
        return res;
    }
}
```
