# [二叉搜索树的第k大节点](https://leetcode-cn.com/problems/er-cha-sou-suo-shu-de-di-kda-jie-dian-lcof/)

### 方法一

```java
class Solution {
    private int res;
    private int count;

    public int kthLargest(TreeNode root, int k) {
        count = k;
        traversal(root);
        return res;
    }

    private void traversal(TreeNode root) {
        if (root == null) {
            return;
        }
        //先序遍历从小到大，反之从大到小
        inTraversal(root.right);
        count--;
        if (count == 0) {
            res = root.val;
        }
        inTraversal(root.left);
    }
}
```

### 方法二

```java
class Solution {
    public int kthLargest(TreeNode root, int k) {
        int res = 0, count = k;
        Deque<TreeNode> stack = new ArrayDeque<>();
        TreeNode p = root;
        while (p != null || !stack.isEmpty()) {
            //先序遍历从小到大，反之从大到小
            while (p != null) {
                stack.push(p);
                p = p.right;
            }
            p = stack.pop();
            count--;
            if (count == 0) {
                return p.val;
            }
            p = p.left;
        }
        return -1;
    }
}
```
